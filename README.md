# Live AI Assistant

A web-based AI assistant built with Node.js and Python.

## Features
- Glassmorphism UI
- Chat interface
- Local AI intent classification model
- Calculator
- Name memory
- Date and time responses

## Tech Stack
- Node.js (backend API)
- Python (AI model)
- Flask
- scikit-learn
- HTML / CSS / JavaScript

## How it Works

User → Web UI → Node.js → Python AI model → Response → Web UI

## Run the Project

Start the AI model:
- cd ai-model
- python app.py


Start the Node server:
- npm start


Open:
- http://localhost:3000
